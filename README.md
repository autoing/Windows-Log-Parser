# Windows-Log-Parser
Windows日志分析工具，自动分析事件ID为4624、4625、4720、4725、4726、1149的安全日志。

# 工具重写，将支持更多日志分析。

1149

改用进度条，直接打印到窗口速度会很慢。

# 使用说明
运行Windows日志提取.bat，将生成的三个文件放到程序同目录下，双击运行工具即可。
